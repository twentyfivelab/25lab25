# 25 Lab.

Site statique premium et prêt à publier pour l'agence digitale **25 Lab.**

- `index.html`
- `creations.html`
- `temoignages.html`
- `devis.html`
- `contact.html`
- `404.html`

- `logo.svg`

- `logo-dark.svg`
- `favicon.svg`

## Direction artistique

- Fond clair, sobre et haut de gamme
- Typographie : `Space Grotesk` pour les titres et `Outfit` pour les textes
- Palette pastel utilisée avec retenue dans les cartes, les accents et les séparations
- Boutons principaux anthracite pour garder un rendu élégant, sérieux et professionnel

## Parcours du site

- Accueil : positionnement, services, bénéfices, méthode, FAQ et CTA
- Créations : formats de sites, services détaillés, inclusions et comparaison
- Témoignages : preuves, retours clients et réassurance
- Devis : audit gratuit, formulaire, méthode et FAQ
- Contact : formulaire, WhatsApp, Instagram et informations pratiques

## Mise en ligne sur GitHub Pages

1. Placez les fichiers à la racine du dépôt.
2. Activez GitHub Pages sur la branche publiée.
3. Si vous utilisez un domaine personnalisé, ajoutez votre fichier `CNAME`.
4. Si le domaine final n'est pas `25lab.fr`, mettez à jour les URLs canoniques, `robots.txt` et `sitemap.xml`.

## Formulaires

Les formulaires sont prêts visuellement et côté validation front-end.

Pour les connecter :

- remplacez `action="#"` par votre endpoint Formspree, Netlify Forms ou votre handler e-mail ;
- adaptez la logique du fichier `script.js` si votre solution d'envoi en a besoin.
